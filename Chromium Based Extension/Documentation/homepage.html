Bibliography Counselor

The server, having no https installed on it, is running on a Free-Tier and will serve till the free limit exhausts. We are trying to switch it to another server soon in its next version.

About Bibliography Counselor

Description: This is a Bibliography Citation Recommendation system integrated with Overleaf.

Note: Because we are using http service now it is necessary to let the browser know `Proceed to Unsafe' site from Console after the Extension has been invoked for the first time OR you can click here to do so simply

Authors: M Mitra, D Pal, D Roy and D Dutta

Permission: Works only on Overleaf space

Data Security:

  1. Does not store any information about the content of the TEX project.

  2. Uses only the .bib file, the query text and

  3. Users' details to keep track of to whom the service is being provided.

Selecting the Query

Assuming NOT in Rich Text editor mode of Overleaf, Select the Query in usual way.

Interacting with the Extension

Simply, press Ctrl + Shift + S (for chrome) to invoke the `Counselor' and configure it as per Users' choice.

However, for Firefox there exists an issue of focus during invoking the Extension by keyboard shortcut which is Ctrl + Shift + U. Invoking it through the icon on the toolbar is a reliable option though.

Outcome

Chosen References should be correctly placed after the selected query within \\cite command having their respective keys as parameters and the BibTeX String should be appended at first encountered .bib file in the project.
